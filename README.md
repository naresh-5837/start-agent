# Zoho Start — LLC Formation Agent

An AI-powered LLC formation assistant built with React + Vite and Claude API (Anthropic).

## Tech Stack

- React 18 + Vite
- Claude API (Sonnet 4 + Haiku 4.5)
- Zoho Catalyst Slate

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
